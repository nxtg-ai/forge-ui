# .asif/ - Personal Design Workspace

This directory is your **personal design playground** for dog-fooding NXTG-Forge.

## What's This For?

**ASIF** = "As-If" - experimental UI/UX explorations and design prototypes that you're testing before integrating into the main product.

## What Goes Here

✅ UI mockups and prototypes
✅ Experimental component designs
✅ Design system explorations
✅ Personal design assets
✅ Work-in-progress UI code
✅ Design iteration logs

## Git Status

🚫 **Fully .gitignored** - Nothing in this directory will ever be committed to the public repo.

This is your safe space to experiment, break things, and iterate on designs without polluting the public codebase.

## Structure Suggestion

```
.asif/
├── UI-DESIGN/          # UI explorations
│   ├── dx3-v2/         # Design iteration v2
│   ├── prototypes/     # Interactive prototypes
│   └── mockups/        # Static mockups
├── components/         # Experimental components
├── experiments/        # One-off experiments
└── assets/            # Design assets (images, fonts, etc.)
```

## Tips

- **Iterate freely** - This is dog-food territory, break things!
- **Keep notes** - Document what works and what doesn't
- **Extract learnings** - When a design is ready, migrate it to `src/`
- **Don't hoard** - Clean out failed experiments periodically

**Remember:** This directory ships EMPTY in the public release. It's your personal workspace.
