name: nxtg-status
description: Display project status and Forge configuration
category: project
---

Show comprehensive project status including:

## Information Displayed

1. **Project Overview**
   - Name and description
   - Technology stack
   - Directory structure

2. **Forge Configuration**
   - Active agents
   - Available commands
   - Enabled hooks
   - Loaded skills

3. **Git Status**
   - Current branch
   - Modified files
   - Commit history

4. **Health Checks**
   - Dependencies status
   - Test coverage
   - Build status
   - Known issues

Use this command to quickly understand the current state of your project and Forge setup.
