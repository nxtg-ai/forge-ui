Analyze and optimize your codebase for performance and maintainability.

## Optimization Areas

1. **Performance**
   - Identify bottlenecks
   - Optimize algorithms
   - Reduce memory usage
   - Improve query efficiency

2. **Code Quality**
   - Remove dead code
   - Reduce duplication
   - Improve naming
   - Simplify complex functions

3. **Dependencies**
   - Update outdated packages
   - Remove unused dependencies
   - Reduce bundle size
   - Security patches

4. **Architecture**
   - Identify coupling issues
   - Suggest refactoring
   - Improve modularity

## Output

- Performance metrics before/after
- Specific optimization suggestions
- Refactoring recommendations
- Updated, optimized code

Make it faster, smaller, cleaner.
