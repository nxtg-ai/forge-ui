Initialize NXTG-Forge v3.0 in the current project.

## What This Does

1. **Detects Project Type**: Analyzes your codebase
2. **Creates .claude Structure**: Adds Forge capabilities
3. **Sets Up Configuration**: Configures for your stack
4. **Initializes Git Hooks**: Integrates with your workflow

## Process

1. Analyze existing project structure
2. Determine technology stack
3. Create .claude directories
4. Add appropriate agents and commands
5. Configure hooks for your workflow
6. Create initial configuration

## Usage

Run this command in your project root to add Forge capabilities.
